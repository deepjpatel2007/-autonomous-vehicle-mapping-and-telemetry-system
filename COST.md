## Project Cost Analysis

The Hybrid Autonomous Navigation, Mapping, and Manual Override Vehicle was intentionally designed as a low-cost robotics platform capable of demonstrating concepts commonly found in significantly more expensive autonomous systems. The project leverages commercially available development hardware, custom-designed components, and additive manufacturing techniques to create a scalable and affordable autonomous robotics platform.

Many of the electronic components used throughout development were sourced from the ELEGOO Super Starter Kit for Arduino, which provides a comprehensive collection of sensors, actuators, prototyping equipment, and supporting hardware. Additional components were acquired separately or manufactured specifically for the project.

### Budget Analysis

One of the primary design goals of this project was to create an autonomous robotics platform using affordable, readily available hardware while maintaining the ability to demonstrate advanced engineering concepts. By utilizing the ELEGOO Super Starter Kit as the foundation of the platform, the project significantly reduces development costs while still providing access to a diverse collection of sensors and electronic components.

Despite a total estimated project cost of approximately **$108 CAD**, the system demonstrates concepts commonly found in significantly more expensive autonomous robotic platforms, including environmental perception, autonomous navigation, obstacle avoidance, digital mapping, embedded control systems, wireless communication, and fault-tolerant operation.

The low-cost nature of the platform makes it highly accessible for experimentation, rapid prototyping, educational use, and future expansion. The modular architecture allows additional sensors, communication modules, and mapping technologies to be integrated without requiring substantial redesign of the existing hardware infrastructure.

### Engineering Value

Although the project cost remains relatively low, it provides hands-on experience with a wide range of engineering disciplines, including:

- Embedded Systems Development
- Autonomous Robotics
- Sensor Integration
- Environmental Mapping
- Wireless Communication
- Control Systems Engineering
- Real-Time Programming
- Mechatronics
- Software Architecture
- Hardware/Software Integration

From an engineering perspective, the educational and technical value of the platform significantly exceeds its material cost, making it an effective portfolio project for demonstrating practical robotics and embedded systems development skills.
